# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-05-13

### ✨ Agregado
- Portafolio inicial con diseño "Minimalismo Estratégico"
- Frontend React 19 con TypeScript
- Backend Express.js
- Paleta de colores personalizada (azul + naranja)
- Animaciones fluidas con Framer Motion
- Componentes UI accesibles con Radix UI
- Tema claro/oscuro automático
- Responsive design para móvil, tablet y desktop
- Documentación completa
- Workflows de CI/CD con GitHub Actions

### 🔧 Configuración
- Vite como bundler principal
- Tailwind CSS para estilos
- TypeScript para seguridad de tipos
- Prettier para formateo consistente
- pnpm como gestor de paquetes
