## 🎯 Descripción

Este proyecto es un portafolio personal que combina **minimalismo estratégico con acentos de energía digital**. Representa la transformación profesional de Luciano, desde sus inicios en marketing digital hasta el desarrollo de software.

### Características Principales

✨ **Diseño Moderno y Responsivo**
- Interfaz limpia con navegación intuitiva
- Completamente adaptable a dispositivos móviles
- Paleta de colores profesional (azul + naranja)

🎨 **Interactividad Avanzada**
- Animaciones suaves y transiciones fluidas
- Componentes UI personalizados con Radix UI
- Tema claro/oscuro automático

📱 **Full-Stack Moderno**
- Frontend: React 19 + TypeScript + Vite
- Backend: Express.js con Node.js
- Deploy listo para producción

🔧 **Calidad de Código**
- TypeScript para seguridad de tipos
- Prettier para formato consistente
- Estructura modular y escalable

## 🚀 Inicio Rápido

### Requisitos Previos

- **Node.js** v20 o superior
- **pnpm** v10.4.1 (gestor de paquetes recomendado)

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/LucianoCMInteligencia/luciano-cv-portfolio.git
cd luciano-cv-portfolio

# Instalar dependencias
pnpm install
```

### Desarrollo

```bash
# Iniciar servidor de desarrollo
pnpm dev

# Acceder a http://localhost:5173
```

### Build y Producción

```bash
# Compilar proyecto
pnpm build

# Iniciar servidor en producción
NODE_ENV=production pnpm start
```

### Otros Comandos

```bash
# Verificar tipos TypeScript
pnpm check

# Formatear código
pnpm format

# Previsualizar build
pnpm preview
```

## 🎨 Filosofía de Diseño

### Concepto: "Minimalismo Estratégico"

Este portafolio refleja tres pilares fundamentales:

1. **Claridad Estratégica** - Cada elemento tiene propósito comunicativo
2. **Energía Controlada** - Movimiento sutil, colores vibrantes pero profesionales
3. **Narrativa Visual** - Cuenta la historia de transformación personal

### Paleta de Colores

- 🔵 **Azul Primario**: #0056B3 (confianza, profesionalismo)
- 🟠 **Naranja Acentos**: #FF8C42 (energía, creatividad)
- ⚪ **Fondo Neutro**: #FFFFFF y #F8F9FA (claridad, respiración visual)

## 🛠 Stack Tecnológico

### Frontend
- **React 19** - UI library moderna
- **TypeScript** - Seguridad de tipos
- **Tailwind CSS** - Estilos utilitarios
- **Vite** - Bundler rápido
- **Radix UI** - Componentes accesibles
- **Framer Motion** - Animaciones fluidas

### Backend
- **Express.js** - Framework HTTP
- **Node.js** - Runtime JavaScript
- **esbuild** - Compilador rápido

### Desarrollo
- **Prettier** - Formateador de código
- **TypeScript Compiler** - Verificación de tipos
- **pnpm** - Gestor de paquetes eficiente
